# Activation

Owner: Product

Activation is the path from first session to first useful food-intent signal.

## Activation Moments

- First search with relevant results.
- First saved post or place.
- First review/post with dish or place context.
- First follow of a useful reviewer.
- First profile signal that reflects taste.

## Current Paths

- Guests can browse before auth.
- Auth prompts appear on meaningful actions.
- Sign-up collects profile details after credentials.
- Create flow prompts for location, ratings, cuisine, best dish, and dish tags.

## Signals

- Search-to-save conversion.
- Guest prompt conversion on save, like, follow, comment, or create.
- Sign-up completion after credential entry.
- First post completion and first save completion.

## Guardrails

- Do not block browsing too early.
- Ask for profile/taste input only when it improves immediate utility.
- Keep onboarding lightweight until content density supports personalization.
