# Search Capabilities

## Capability Matrix

| Capability | Status | Owner |
| --- | --- | --- |
| Dish search | Supported | Search |
| Place search | Supported | Search |
| Collection search | Supported behind rollout | Search |
| Post search | Supported | Search |
| People search | Supported | Search |
| Occasion discovery | Supported as discovery intent | Product/Search |
| Taxonomy filtering | Supported via accepted assignments only | Search/Taxonomy |
| Query aliases | Supported via taxonomy-owned aliases | Taxonomy |
| Saved searches | Supported behind rollout | Client/Search |
| Nearby sorting | Supported with explicit location source | Search |
| Price filtering | Not supported in v1 | Product |
| Rating filtering | Not supported in v1 | Product |
| Saved-count filtering | Not supported in v1 | Product |
| Yelp-style advanced filters | Not supported in v1 | Product |
| LLM query interpretation | Not supported in v1 | Product |
| AI chatbot search | Not supported | Product |
| Reservations | Not supported | Product |
| Menu database | Not supported | Product |

## Non-Goals

- Restaurant reservation engine
- Menu database
- Yelp-style faceted search
- Full-text review search as the main experience
- AI chatbot search
- LLM query interpretation
- Infinite filter expansion
- Price, rating, saved-count, or advanced filters in v1
