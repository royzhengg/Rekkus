# Funnels

Funnels own the product journeys Rekkus measures for activation, discovery, contribution, and retention.

## Priority Funnels

| Funnel | Steps | Owner |
| --- | --- | --- |
| Search to save | Search query -> result click -> restaurant view -> save | [../../product/SEARCH.md](../../product/SEARCH.md) |
| Discover to intent | Places/search view -> detail view -> save or post view | [../../product/DISCOVERY.md](../../product/DISCOVERY.md) |
| Create review | Start create -> choose place -> add media/details -> publish | [../../product/CONTRIBUTION_LOOPS.md](../../product/CONTRIBUTION_LOOPS.md) |
| Auth conversion | Guest/read -> auth gate -> sign up/login -> completed write | [../../product/ACTIVATION.md](../../product/ACTIVATION.md) |
| Retention | Save -> return -> revisit/review/comment | [../../product/RETENTION.md](../../product/RETENTION.md) |
| Share to friend via DM | Post detail -> Share -> Select conversation -> Send -> Recipient opens app -> Views shared post -> Saves | [../../product/MESSAGING.md](../../product/MESSAGING.md) |
| Group chat creation | Tap "New group" -> Select members (2+) -> Set name -> Create -> Send first message -> All members open conversation | [../../product/MESSAGING.md](../../product/MESSAGING.md) |

## Rules

- Funnels should explain drop-off in food utility, not just generic conversion.
- Do not add tracking to sensitive account/privacy flows without a compliance review.
- Funnel metadata must use categories and booleans, not free-form private user data.
